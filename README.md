# Guess The Celebrity
An Android Application developed using Java that tests your knowledge of Hollywood celebrities.  

It uses data from the Swedish site [Posh24](http://www.posh24.se/kandisar) and extracts the celebritiesâ€™ names & images directly from the page source.
